# Tier 0 Self-Check

Do this without looking at the source.

## Build
1. Implement a two-input neuron with a step activation.
2. Hand-set weights and bias for AND.
3. Hand-set weights and bias for OR.
4. Train AND with the perceptron update.
5. Try XOR and show that it fails.

## Explain
- Draw the decision line for AND.
- Explain what the bias does.
- Explain why XOR cannot be split by one line.

## Pass Condition
You can rebuild the neuron and perceptron rule from memory, and you can explain
XOR failure geometrically.
