# Learning System

This folder turns the lab into a self-training loop.

Use it like this:

1. Run an experiment.
2. Fill a checkpoint from `templates/checkpoint.md`.
3. Answer the matching questions in `questions/`.
4. Do the tier self-check without looking at code.
5. Rewrite one derivation in `infobase/04_derivations/` in your own words.

The goal is not to collect notes. The goal is to prove, repeatedly, that you can
reconstruct the idea from memory.
